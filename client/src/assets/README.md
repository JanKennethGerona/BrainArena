# Assets Directory

Store your static assets here:

## Images
- `images/` - Brand logos, icons, photos, backgrounds, etc.

## How to Use Images in Components

```jsx
// Import the image
import logo from '../assets/images/logo.png'

// Use it in your JSX
<img src={logo} alt="BrainArena Logo" />
```

## Recommended Image Organization

```
assets/
  images/
    branding/     - Logos, brand assets
    icons/        - UI icons
    backgrounds/  - Background images
    avatars/      - User avatars
```
